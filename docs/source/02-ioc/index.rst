IoC-контейнер
=============

Современные приложения состоят из настраиваемых, взаимосвязанных **компонентов**. Чем сложнее логика приложения,
тем сложнее компоненты и связи между ними. Ключом успешной разработки современных приложений является принцип
написания слабо связанного кода, суть которого в том, что каждый компонент системы должен быть как можно более
изолирован и не должен полагаться в своей работе на детали конкретной реализации других компонентов. Слабосвязанные
приложения являются более гибкими и легкими в обслуживании. Они также проще поддаются проверке во время разработки.

Все компоненты приложения InfinniPlatform управляются с помощью встроенного IoC_-контейнера, который позволяет
упростить и автоматизировать написание слабо связанного кода. IoC-контейнер хранит список компонентов приложения,
автоматически определяет связи между ними и осуществляет контроль времени жизни каждого компонента.
